Detailed Description

When Mbed Crypto is built with the IOTEX_PSA_CRYPTO_SPM option enabled, this header is included by all .c files in Mbed Crypto that use PSA Crypto function names. This avoids duplication of symbols between TF-M and Mbed Crypto.

Note
This file should be included before including any PSA Crypto headers from Mbed Crypto.
