922 /* In the encoding of a MAC algorithm, the bits corresponding to
923  * PSA_ALG_MAC_TRUNCATION_MASK encode the length to which the MAC is
924  * truncated. As an exception, the value 0 means the untruncated algorithm,
925  * whatever its length is. The length is encoded in 6 bits, so it can
926  * reach up to 63; the largest MAC is 64 bytes so its trivial truncation
927  * to full length is correctly encoded as 0 and any non-trivial truncation
928  * is correctly encoded as a value between 1 and 63. */
