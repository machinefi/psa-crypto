926  * PSA_ALG_MAC_TRUNCATION_MASK encode the length to which the MAC is
927  * truncated. As an exception, the value 0 means the untruncated algorithm,
928  * whatever its length is. The length is encoded in 6 bits, so it can
929  * reach up to 63; the largest MAC is 64 bytes so its trivial truncation
