Detailed Description

Cipher information. Allows calling cipher functions in a generic way.

Note
The library does not support custom cipher info structures, only built-in structures returned by the functions iotex_cipher_info_from_string(), iotex_cipher_info_from_type(), iotex_cipher_info_from_values(), iotex_cipher_info_from_psa().
